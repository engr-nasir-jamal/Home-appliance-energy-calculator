# Home Appliance Energy Cost Calculator

A Python application that calculates the daily and monthly energy consumption of household appliances and estimates their daily and monthly electricity cost.

The program allows users to:
- Select appliances from a CSV dataset
- Enter appliance quantity and daily usage hours
- Calculate daily and monthly energy consumption
- Calculate daily and monthly electricity cost
- View total household energy usage and cost
- Identify the appliance with the highest monthly energy consumption
- Export the final energy report as a JSON file

## How to Use
- Run th program
- Input appliances by entering the appliance respective number
- Input quantity of appliances and their working hours per day
